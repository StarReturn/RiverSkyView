# 江天一览 RiverSkyView 完整全套品牌资料

## 一、文化溯源（双重底蕴，无神话、不仿古帝王玄幻）

### 1. 诗词本源：《春江花月夜》核心句

原文：**江天一色无纤尘，皎皎空中孤月轮**抖音百科

释义：江水长空融为一体，澄澈通透，没有一丝尘埃；天地视野开阔、万物尽收眼底。

匹配产品内核：软件如同一片通透江天，电脑全磁盘分散项目统一全景展示，界面干净无冗余。

### 2. 历史意象：金山「江天一览」御题文脉

康熙南巡登镇江金山，题写**江天一览**石碑，立于山顶江天一览亭，寓意登高望远、尽收千里江川全景。

意象：登高俯瞰整片江河大地 → 软件一站式俯瞰电脑全部本地工程目录，完美契合 GUI 全景视图核心需求。

### 3. 风格优势

- 无龙凤、天宫、上古神兽、玄宸等华为式古风神话词；
- 意象是**江、天、远眺、澄明、收纳、全景**，山水清雅开阔；
- 和之前「域铸」冶金重工意象完全区分，差异化极强。

## 二、中英文名称解析

### 中文名：江天一览

1. **江**：指代电脑本地磁盘所有分区、层层嵌套的项目文件夹（江河绵延万千分支）；
2. **天**：全局桌面工作台、完整可视化视图、所有开发资源的全域空间；
3. **一览**：核心功能 —— 一窗尽收全部项目，文件树、图片 / MD 预览、操作日志日历、密钥文档全部统一浏览。

### 英文名：RiverSkyView

- River = 江，对应本地项目目录集群；

- Sky = 天，全域工作台；

- View = 览，可视化 GUI 视图，精准点明桌面可视化工具属性；

  

  优势：单词简短，拼写流畅，软件窗口标题、托盘图标、安装包命名简洁，海外 / 开源场景通用。

## 三、与你的需求功能一一对应释义

1. 多项目分散文件夹统一收纳

   

   江川千万支流汇于一片江天，所有散落 D/E 盘项目归集到软件列表，不用逐层翻文件夹。

2. GUI 文件树、图片 / MD 预览、全景视图

   

   「一览」登高望远，全屏可视化浏览项目文件，支持图片缩放旋转、文档在线阅读。

3. 多标签 CMD 终端、项目快速进入

   

   江天之下随心泛舟，任意项目一键唤起终端，自动复用标签页，无需重复打开资源管理器。

4. pm_log 时序日志、操作热力日历

   

   江月循环往复，以时间维度记录每日操作，生成类 GitHub 提交日历，追溯项目修改轨迹。

5. 独立密码 TXT 文档仓库

   

   江天之内自有藏阁，单独存储密钥文档，支持新建、预览、编辑、软删除。

6. 项目本地标记防重复添加、软移除管理

   

   江界分明，每个项目自带专属标记，收纳有序，移除仅隐藏不删除本地源码。
